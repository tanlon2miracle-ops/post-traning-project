# 🔵 开发/工程工作区

开发和工程同学的主阵地。

## 目录说明

| 目录 | 用途 | 说明 |
|------|------|------|
| `envs/` | 环境配置 | H20 集群环境、Dockerfile、requirements.txt、安装脚本 |
| `data_pipeline/` | 数据管道 | Data-Juicer 清洗配置、格式转换脚本、标签映射逻辑 |
| `deployment/` | 部署脚本 | vLLM 推理部署、压测工具、监控配置 |

## 核心交付物

1. **环境搭建文档** — H20 集群上装好 LLaMA-Factory / vLLM / Data-Juicer
2. **数据管道脚本** — 原始数据 → 清洗 → 格式化 → 训练就绪
3. **部署脚本** — 训练好的模型一键部署推理服务
