# 🟢 运营/风控工作区

运营和风控策略同学的主阵地。

## 目录说明

| 目录 | 用途 | 说明 |
|------|------|------|
| `rules/` | 结构化规则库 | JSON/YAML 格式，供数据合成和训练脚本读取 |
| `sample_data/` | 脱敏样例数据 | **极少量**（10-20条），供开发调试用，严禁放真实数据 |
| `taxonomy_updates/` | 标签体系变更提案 | 需要调整标签时在此提交提案 |

## ⚠️ 数据安全

- `sample_data/` 只放**严格脱敏**的极少量样例
- 真实人审数据放 `data/`（已 gitignore，不入库）
- 规则库如涉及敏感策略，评估后再决定是否入库
